DirectX11
3D Game Engine

[Timeline: 2021.08 - 2022.02] | Role: Independent Developer

A modern 3D game engine built on DirectX 11 and Win32 interfaces, featuring advanced rendering capabilities and robust architecture. The engine implements contemporary game engine subsystems including Resource Management, Job System, and Render Graph, with an emphasis on low coupling and high extensibility.

Project Screenshot - Click to enlarge

Advanced Rendering

Support for multiple rendering techniques including Phong Shading, Cel Shading, and Percentage Close Soft Shadows

Development Timeline

Foundation (Aug 2021)

Win32 Window system with mouse/keyboard IO and basic exception handling. Initial Phong Shading implementation.

Core Systems (Sep-Oct 2021)

Drawable and Bindable systems for automatic rendering process assembly. Bindable Pool system for efficient resource management.

Advanced Features (Nov-Dec 2021)

Dynamic Vertex System and Dynamic Constant System. Integration with AssImp and DirectXTex.

Final Architecture (Jan-Feb 2022)

Job System for multi-pass rendering and Render Graph System using DAG structure.

Engine Interface with ImGui Integration
